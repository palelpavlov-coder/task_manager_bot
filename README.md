# 📝 Telegram Task Manager Bot

Асинхронный Telegram-бот на Python для удобного управления списком задач и заметок.

## 🚀 Основные возможности
* Добавление задач: Просто отправьте боту любой текст, и он добавит его в список.
* Просмотр списка (`/tasks`): Бот выведет нумерованный список всех задач.
* Подсчет задач (`/count`): Быстрая команда для получения общего количества дел.
* Очистка списка (`/clear`): Удаление всех записей одной командой.

## 🛠 Технологический стек
* Python 3.10+
* Aiogram 3.x — современная библиотека для работы с Telegram Bot API.
* Python-dotenv — инструмент для безопасного хранения токенов.
